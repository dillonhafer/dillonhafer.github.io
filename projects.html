Budgetal

Monthly budgeting software

  2013-{{ site.time | date: '%Y' }}

This is just one of my many personal side projects. The application is for organizing a monthly, zero-sum budget. I was looking for financial software with specific features, but couldn't find one to fit my needs. I built this application to solve my problem. Using a Ruby on Rails JSON api, the application uses ReactJS for the UI to provide a very responsive and snappy experience.
